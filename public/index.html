Version 1.5.4 — "Zenith Hotfix 4" (July 15, 2026)

▼

Release Notes:

  • UI & Styling Refinements: Cumulative update incorporating visual contrast improvements on the web portal.
  • Data Export Polish: Finalized stability fixes for background archive downloads.

Version 1.5.3 — "Zenith Hotfix 3" (July 15, 2026)

▼

Release Notes:

  • Data Export Silent Failure Fix: Fixed a critical bug where data exports would queue in DownloadManager but silently fail to download. The app now correctly resolves the relative path returned by the server into a fully-qualified domain URL, enabling stable `.zip` downloads.

Version 1.5.2 — "Zenith Hotfix 2" (July 15, 2026)

▼

Release Notes:

  • Data Export Download Fix: Replaced the legacy external browser intent with native Android `DownloadManager`. The `.zip` archive now downloads securely and directly into the device's Downloads folder without crashing on devices lacking default browser configurations.

Version 1.5.1 — "Zenith Hotfix" (July 15, 2026)

▼

Release Notes:

  • Immediate Startup Crash Resolution: Reverted R8 code minification (`isMinifyEnabled`) and resource shrinking to restore 100% reflection and bytecode compatibility (`v1.4.0` standard), ensuring clean launch and stability.
  • Version & Signature Conflict Fix: Bumped version to `1.5.1` (version code `12`) to resolve Google Play package installer conflicts and ensure smooth upgrades over all prior builds.

Version 1.5.0 — "Zenith" (July 15, 2026)

▼

Release Notes:

  • Production Firestore Security Lockdown: Enforces strict user-isolated access control (`firestore.rules`) safeguarding ride histories, emergency contacts, and feedback against unauthorized access.
  • PostHog Product Telemetry & Remote Kill-Switch: Privacy-aware event tracking across authentication, ride lifecycle, and SOS beacons, dynamically controlled via Firebase Remote Config.
  • Centralized UI Theming & Chart Polish: Standardized color design tokens across core screens and improved edge avoidance and padding on scrubbable telemetry charts.
  • Build & ProGuard Optimization: Minified and resource-shrunk release bundle configuration for peak runtime performance and security.

Version 1.4.0 — "Summit" (July 12, 2026)

▼

Release Notes:

  • Complete Data Ownership & Cloud Export: Download your entire tracking history from the cloud in a secure, tokenized on-demand ZIP flow to prevent vendor lock-in.
  • Live Ride Sharing with Notifications: Share your real-time ride progress with friends or family via a secure live-tracking link, complete with interactive red-dot UI notifications and personalized SMS templates.
  • Emergency SOS Refactor: Streamlined emergency beacon trigger to an intuitive push-button action with smooth visual feedback.
  • Smart Graph Edge Avoidance: Telemetry graphs intelligently flip labels to prevent any data from being clipped at the screen edges during interaction.

Version 1.3.0 — "Vanguard" (July 12, 2026)

▼

Release Notes:

  • Interactive Live Share Pill Drawer: Sleek upward glassmorphism drawer (90% transparency) with dynamic state animations, clean symbology, one-click X transform, and crisp haptics.
  • Orphaned Ride Auto-Recovery: Intelligent trajectory recovery that automatically reconstructs complete ride stats (distance, speed, duration) from persisted SQLite GPS points if the device powers off or app force-closes mid-ride.
  • Completed Rides Isolation & Delete Guard: History screen strictly lists completed sessions while safeguarding active rides against accidental deletion.
  • LTTB Graph Downsampling: High-fidelity telemetry chart rendering that preserves visual peaks and uniform spacing for any duration without spikiness.

Version 1.2.1 — "Ascent" (July 9, 2026)

▼

Release Notes:

  • In-App Auto-Update Notifications: Asynchronously checks for new updates via cloud configuration on launch and presents an interactive release notes dialog with one-click store updating.
  • Manual Update Check: Added a dedicated "Check for Updates" button on the Settings screen.
  • CI/CD & Security Hardening: Automated JSON schema resilience and API key configuration hardening.

Version 1.2.0 — "Horizon" (July 9, 2026)

▼

Release Notes:

  • Ride History UX Overhaul: Compact high-density cards with native date sticky headers and vector route thumbnails.
  • Smart Background Cloud Sync: Automated daily background synchronization with WorkManager and lazy list loading.
  • Global Multi-Language Localization: Full translation support across 7 languages (English, Spanish, French, German, Hindi, Japanese, Chinese).

Version 1.1.0-rc.5 (June 27, 2026)

▼

Release Notes:

  • Settings & Account Management Redesign: Split the settings into a cleaner interface with a dedicated Account Management screen for privacy controls.
  • Enhanced Account Deletion Security: Added explicit typing confirmation before permanently deleting accounts to prevent accidental data loss.
  • Improved Export Preview: Refined the visual layout of the Image Export screen and added a new "Save to Gallery" button.

Version 1.1.0-rc.4 (June 26, 2026)

▼

Release Notes:

  • Live GPS Status: Added a visual warning banner when GPS signal is lost during an active ride.
  • Smart Ride Discarding: Rides that fail to capture any GPS points are now automatically discarded to keep your history clean.
  • Enhanced GPX Import: The "Import GPX" button is now clearer and includes validation to ensure only valid .gpx files are imported.
  • Bug Fixes: Resolved layout issues and crashes on the Ride History and Detail screens.

No release for iOS yet. We are working hard to bring TrackMe to the App Store soon. Stay tuned!
